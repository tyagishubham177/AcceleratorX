# Manual upload staging area

This folder is intentionally permanent.

Connected AI tools may be able to edit repository text but may not be able to upload PDFs, DOCX files, spreadsheets, images, ZIP archives, or other binary files. Upload such files here through the GitHub interface first.

After upload, an AI agent or contributor should:

1. identify the intended assignment and file purpose;
2. rename the file using lowercase kebab-case where practical;
3. move it into the appropriate `assignment-01`, `assignment-02`, `assignment-03`, `submission`, or `evidence` folder;
4. update the relevant README links; and
5. remove the staged copy from this folder.

Do not treat this folder as disposable, delete its README, or use it as permanent storage for completed assignment artefacts.
