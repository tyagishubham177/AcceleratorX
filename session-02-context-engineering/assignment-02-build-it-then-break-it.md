# Assignment 2: Build It, Then Break It

**Session:** Context Engineering  
**Status:** Not started

## Goal

Design a layered production-assistant prompt, red-team it, patch the failures, and separate prompt rules from hard-coded guardrails.

## Required work

- [ ] Define the assistant scenario.
- [ ] Write a v1 system prompt with labeled layers:
  - identity/persona
  - scope and capabilities
  - hard constraints
  - business-policy constraints
  - style
  - output format
- [ ] Test all 10 attack categories:
  - direct injection
  - role-play jailbreak
  - prompt extraction
  - scope creep
  - fabricated authority
  - urgency/social engineering
  - data exfiltration
  - emotional pressure
  - obfuscation
  - multi-turn erosion with at least six turns
- [ ] Maintain a specific break log with evidence and severity.
- [ ] Create v2 targeted at logged v1 failures.
- [ ] Re-test every attack that broke v1.
- [ ] Compare v1 and v2 changes.
- [ ] Analyse two constraints as prompt rules versus external guardrails.
