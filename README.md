# AcceleratorX: Prompt Engineering Assignments

A session-wise repository for my AcceleratorX work across prompt engineering, context engineering, red-teaming, evaluation, and prompt tracking.

## For peers and visitors

This repository is intended as a reference for AcceleratorX peers and others exploring practical prompt engineering.

You will find assignment briefs, prompts, raw model outputs, evaluation tables, testing logs, screenshots, and conclusions as the work is completed.

> This is an independent participant repository, not an official AcceleratorX repository. Please use it for reference rather than copying submissions.

## Sessions

| Session | Topic | Assignment | What you will find |
|---|---|---|---|
| 01 | Prompt Engineering Foundations | [The Technique Ladder](./session-01-prompt-engineering-foundations/assignment-01-technique-ladder.md) | A comparison of multiple prompting techniques on the same task |
| 02 | Context Engineering | [Build It, Then Break It](./session-02-context-engineering/assignment-02-build-it-then-break-it.md) | A layered system prompt, adversarial tests, fixes, and guardrail analysis |
| 03 | Prompt Lab + LangSmith | [Prompting as an Engineering Practice](./session-03-prompt-lab-langsmith/assignment-03-prompting-as-engineering-practice.md) | Versioned prompt iterations, evaluation results, and run tracking |

Each session has its own folder and assignment file. Supporting artefacts will be added alongside them where relevant.
