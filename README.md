## Repository structure

Each assignment follows: `README.md` -> official brief -> `solution.md` -> `submission/` -> `evidence/`.

The active verification configuration is **`gpt-5.6-luna`**, reasoning effort **medium**, temperature **0.2**, using `POST /v1/responses`.

Shared configuration, source matrix, runner, and manifest are under [`verification/`](./verification/).

## Repository maintenance

`manual-upload/` is a permanent staging area for binary files uploaded through GitHub when connected AI tools cannot upload them directly. See [`manual-upload/README.md`](./manual-upload/README.md).
