# Assignment 3: Prompting as an Engineering Practice

**Session:** Prompt Lab + LangSmith Tracking  
**Status:** Not started

## Goal

Improve one prompt through three measured and versioned iterations, using evidence rather than subjective preference.

## Required work

- [ ] Write a fake employee-policy one-pager with 10–15 facts.
- [ ] Define a 3–5 criterion scoring rubric before v1.
- [ ] Create one fixed eight-question evaluation set containing:
  - at least two easy questions
  - at least two ambiguous questions
  - at least two adversarial questions
  - at least two questions not answered by the policy
- [ ] Write and run system prompt v1 against all eight questions.
- [ ] Score and log all v1 runs.
- [ ] Create v2 targeting specific v1 failures.
- [ ] Re-run the complete evaluation set and score v2.
- [ ] Create v3 targeting specific v2 failures.
- [ ] Re-run the complete evaluation set and score v3.
- [ ] Track version, prompt, model parameters, timestamp, and per-question scores in LangSmith or an equivalent sheet.
- [ ] Compare v1, v2, and v3 by criterion.
- [ ] Document one felt improvement that did not improve the measured score, or caused regression.
- [ ] Define an evaluation gate for shipping v4.
